Personalized Diet Recommendation System
Constraint Satisfaction & Heuristic Search Based Diet Planner
This project is a Personalized Diet Recommendation System that generates optimal meal plans based on user biometrics such as age, weight, height, and diet preference (Veg / Non-Veg).
The system uses Constraint Satisfaction logic and a Heuristic Search Algorithm to select the best meal from a structured food database.

Unlike traditional calorie calculators, this system considers age-based metabolic differences and searches for the most suitable meal option instead of giving fixed diet charts.
